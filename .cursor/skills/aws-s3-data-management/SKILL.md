# AWS S3 Data Management

## Overview

This project uses **AWS S3** for storing and sharing data files.

## Comprehensive Guide

**📖 For Coauthors**: See the comprehensive guide: [`doc/aws_data_access_guide.md`](doc/aws_data_access_guide.md)

This guide covers:
- Install and configure AWS CLI
- Set up AWS credentials and profiles
- Install R packages for S3 access
- Load RDS and CSV files from S3
- Write files to S3
- Common operations and troubleshooting

## Credentials

**Contact the project owner to obtain AWS credentials**:
- Access Key ID
- Secret Access Key
- Bucket name
- Region

## Common Operations

- Load RDS files from S3
- Load CSV files from S3
- Write RDS files to S3
- Write CSV files to S3
- List S3 bucket contents
- Verify AWS configuration
