# Literature Review Skill — Examples

## Example structure (short)

```latex
\subsection{Paper Title in Full (Author1, Author2, \& Author3, Year)}

\begin{itemize}
  \item \textbf{Background:} One sentence on why the question matters.
  \item \textbf{Setting:} Agents, exogenous/endogenous variables, timing.
  \item \textbf{Problem:} Research question or decision.
  \item \textbf{Methodology:} Model type, data (if any), identification in one line.
  \item \textbf{Perspective:} Whose objective (e.g., seller profit vs.\ social welfare).
  \item Main results and intuition:
    \begin{itemize}
      \item \textbf{Result (Proposition X):} Statement. Intuition: ...
      \item \textbf{Result (Proposition Y):} Statement. Intuition: ...
    \end{itemize}
  \item Managerial implication: ...
\end{itemize}
```

## Reference: review.tex format

- Janakiraman et al. (2015): meta-analysis; short summary then dimensions/findings, then moderator/implication.
- Free returns Denmark: background, setting (quasi-experiment), data, methodology, main findings (nested list), economic interpretation, managerial implication.
- Najafi et al. (2025): short summary (policy, setting, perspective), then main results and intuition (nested), managerial implication, perspective bullet.

New entries should follow the same order: short summary (including perspective) then key results and intuition (more detail).
