# Secret reserve price auction

LaTeX draft for the secret-reserve auction model (modular source under `draft/`).

## Build

From `draft/`:

```bash
pdflatex main.tex
pdflatex main.tex
```

Output: `draft/main.pdf`.

## Layout

| Path | Contents |
|------|----------|
| `draft/main.tex` | Master document |
| `draft/sections/` | Body sections and appendices |
| `auction_secret_reserve_price.pdf` | Original reference PDF |
